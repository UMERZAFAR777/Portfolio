# Clone and Project

## Project Description
This project is a front-end clone of the Netflix web interface, created using HTML and CSS and also morden python django. It showcases a modern, responsive design and includes features like an interactive image slider to display popular movies and TV shows. The project demonstrates skills in web layout, styling, and responsive design.

## Features
- Fully responsive UI built with HTML and CSS
- Smooth transitions and animations
- Interactive image slider showcasing content
- Flexbox and grid layout system for organized structure
- Clean and minimalistic design

## Technologies Used
- HTML5
- CSS3 (Flexbox and Grid)
- Bootstrap (for styling and responsiveness)
- Python 
- Django 
## Purpose
This project was created as a practice exercise to improve web development skills and better understand the structure and design of a popular streaming service interface.
